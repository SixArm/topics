# Extreme Programming (XP)

Question: Which of the following best describes the core set of values that guide the Extreme Programming (XP) methodology?

- [ ] Speed, automation, documentation, hierarchy, and isolation
- [ ] Communication, simplicity, feedback, courage, and respect
- [ ] Planning, testing, coding, deploying, and monitoring
- [ ] Agility, scalability, security, performance, and reliability

<details>
  <summary>Answer</summary>
  <p>Communication, simplicity, feedback, courage, and respect</p>
  <p>These five values form the foundational principles of XP. They guide all practices in the methodology: communication ensures team alignment, simplicity focuses on essential solutions, feedback enables rapid iteration, courage supports making necessary changes, and respect fosters healthy team dynamics. Understanding these values is essential because all XP practices—such as pair programming, test-driven development, and continuous integration—are designed to reinforce and embody these core values.</p>
</details>
