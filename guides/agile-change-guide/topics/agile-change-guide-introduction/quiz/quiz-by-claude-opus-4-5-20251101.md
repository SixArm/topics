# Agile Change Guide Introduction

Question: What is the primary purpose of shared context and clear communication in agile teams, as described in the Agile Change Guide?

- [ ] To reduce the need for documentation and formal processes
- [ ] To eliminate the role of project managers in agile environments
- [ ] To accelerate collaboration and help teams forge better project plans
- [ ] To standardize all agile terminology across different industries

<details>
  <summary>Answer</summary>
  <p>To accelerate collaboration and help teams forge better project plans</p>
  <p>The guide emphasizes that shared context and clear communication—described metaphorically as "singing from the same songbook"—helps teams work together more effectively and create better project plans. This is achieved through common understanding of agile terminology and having accessible definitions and explanations.</p>
</details>
