# AI + govtech

Question: What is a key capability of AI in government technology that enables agencies to support evidence-based policymaking?

- [ ] Replacing all human decision-makers with autonomous AI systems
- [ ] Automating data integration, cleaning, and analysis to extract meaningful information
- [ ] Eliminating the need for citizen feedback in policy development
- [ ] Restricting data access to improve security protocols

<details>
  <summary>Answer</summary>
  <p>Automating data integration, cleaning, and analysis to extract meaningful information</p>
  <p>AI's Data Insights capability allows government agencies to automate the complex processes of data integration, cleaning, and analysis. This enables agencies to efficiently extract meaningful information from large datasets, which directly supports evidence-based policymaking by providing actionable insights derived from comprehensive data analysis.</p>
</details>
