# Web Content Accessibility Guidelines (WCAG)

Question: What are the four core principles of the Web Content Accessibility Guidelines (WCAG)?

- [ ] Responsive, Scalable, Secure, and Maintainable
- [ ] Perceivable, Operable, Understandable, and Robust
- [ ] Accessible, Usable, Testable, and Portable
- [ ] Visual, Auditory, Cognitive, and Motor

<details>
  <summary>Answer</summary>
  <p>Perceivable, Operable, Understandable, and Robust</p>
  <p>WCAG is built around four core principles, often remembered by the acronym POUR: Perceivable (users can perceive content), Operable (users can interact with all functionality), Understandable (content and operation are clear), and Robust (content works reliably across different user agents and assistive technologies).</p>
</details>
