# Resource leveling

Question: What is the primary goal of resource leveling in project management?

- [ ] To maximize the number of resources assigned to each task
- [ ] To ensure resources are used efficiently while keeping the project on track
- [ ] To reduce the total project budget by eliminating resources
- [ ] To assign all resources to work simultaneously on every activity

<details>
  <summary>Answer</summary>
  <p>To ensure resources are used efficiently while keeping the project on track</p>
  <p>Resource leveling is a technique that adjusts the project schedule to optimize the use of available resources (people, equipment, materials) so that no one is overburdened or idle. The goal is to balance workloads, ensure efficient and effective resource utilization, and complete the project on time and within budget.</p>
</details>
