# Business data analysis

Question: What is the correct sequence of steps in the business data analysis process?

- [ ] Data visualization → Data collection → Data cleaning → Data interpretation
- [ ] Data collection → Data cleaning → Data exploration → Data visualization
- [ ] Data modeling → Data collection → Data exploration → Data cleaning
- [ ] Data interpretation → Data exploration → Data visualization → Data collection

<details>
  <summary>Answer</summary>
  <p>Data collection → Data cleaning → Data exploration → Data visualization</p>
  <p>The business data analysis process follows a logical sequence: first, data is collected from various sources; then it is cleaned to remove errors and inconsistencies; next, data exploration uses statistical methods to identify patterns and trends; and finally, data visualization creates charts and graphs to communicate insights. This sequence ensures data quality before analysis and effective communication of results.</p>
</details>
