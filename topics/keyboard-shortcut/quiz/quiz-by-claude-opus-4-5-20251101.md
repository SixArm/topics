# Keyboard shortcut

Question: What is a primary benefit of using keyboard shortcuts in software applications?

- [ ] They require extensive training before any use
- [ ] They eliminate the need for a keyboard entirely
- [ ] They accelerate workflow by reducing reliance on mouse navigation
- [ ] They only work in command-line interfaces

<details>
  <summary>Answer</summary>
  <p>They accelerate workflow by reducing reliance on mouse navigation</p>
  <p>Keyboard shortcuts significantly speed up workflows by eliminating the need to navigate through menus or use the mouse for every action. This allows users to perform tasks more efficiently, increasing productivity by reducing time spent on repetitive mouse movements and clicks.</p>
</details>
