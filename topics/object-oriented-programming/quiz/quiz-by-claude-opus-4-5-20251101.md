# Object-oriented programming (OOP)

Question: Which OOP principle involves creating new classes derived from existing classes to inherit their properties and behaviors?

- [ ] Encapsulation
- [ ] Polymorphism
- [ ] Inheritance
- [ ] Abstraction

<details>
  <summary>Answer</summary>
  <p>Inheritance</p>
  <p>Inheritance is the practice of creating new classes that are derived from existing classes, allowing them to inherit properties and behaviors. This enables code reuse and helps create a hierarchy of classes that can model complex systems.</p>
</details>
