# Project Management for the Unofficial Project Manager by Kory Kogon et al.

Question: According to Kogon et al., what are the five core project management processes that every unofficial project manager should understand?

- [ ] Research, development, testing, deployment, and maintenance
- [ ] Initiation, planning, execution, monitoring, and closure
- [ ] Budgeting, staffing, scheduling, reporting, and auditing
- [ ] Brainstorming, prototyping, iteration, launch, and review

<details>
  <summary>Answer</summary>
  <p>Initiation, planning, execution, monitoring, and closure</p>
  <p>The book covers these five essential project management processes as the core framework that guides projects from start to finish. Understanding each phase—from initiating and planning a project, through executing and monitoring progress, to properly closing it out—provides the foundation that unofficial project managers need to successfully deliver results even without formal PM training.</p>
</details>
