# About the ebook PDF

Question: What program is used to convert the merged markdown files into the final PDF output?

- [ ] Adobe Acrobat
- [ ] LaTeX
- [ ] pandoc
- [ ] markdown-text-to-link-urls

<details>
  <summary>Answer</summary>
  <p>pandoc</p>
  <p>The ebook PDF is generated using pandoc, which converts the merged markdown files into PDF format. The repository includes a custom tool called pandoc-from-markdown-to-pdf that uses preferred pandoc settings to add a table of contents, fonts, highlighting, sizing, and more.</p>
</details>
