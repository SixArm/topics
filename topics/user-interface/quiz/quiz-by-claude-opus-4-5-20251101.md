# User Interface (UI)

Question: What is the primary purpose of maintaining consistency in user interface design?

- [ ] To reduce the cost of software development
- [ ] To help users understand how to interact with the system
- [ ] To comply with government regulations
- [ ] To maximize the use of screen real estate

<details>
  <summary>Answer</summary>
  <p>To help users understand how to interact with the system</p>
  <p>Consistency in UI design involves using established design systems, style guides, pattern libraries, conventions, and industry standards. This consistency helps users understand how to interact with the system because familiar patterns reduce cognitive load and allow users to apply prior knowledge to new interfaces, making the experience more intuitive and efficient.</p>
</details>
