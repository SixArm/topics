# Make value flow without interruptions

Question: According to SAFe Principle #6 "Make value flow without interruptions," what does Lean Thinking require organizations to understand in order to achieve continuous value delivery?

- [ ] The budgeting cycles and fiscal year planning processes
- [ ] The properties of a flow-based system and how they accelerate or impede value flow
- [ ] The individual performance metrics of each team member
- [ ] The competitor analysis and market positioning strategies

<details>
  <summary>Answer</summary>
  <p>The properties of a flow-based system and how they accelerate or impede value flow</p>
  <p>SAFe Principle #6 emphasizes that making value flow without interruptions requires understanding what flow is, the various properties of a flow system, and how these properties can accelerate or impede value delivery. The principle highlights eight common properties of a flow-based system and provides recommendations for eliminating impediments to flow.</p>
</details>
