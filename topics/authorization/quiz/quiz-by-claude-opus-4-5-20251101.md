# Authorization

Question: What is the primary purpose of authorization in computer security?

- [ ] To encrypt data transmitted between systems
- [ ] To verify the identity of a user or system
- [ ] To determine whether a user has permission to access a resource or perform an action
- [ ] To detect and prevent malware infections

<details>
  <summary>Answer</summary>
  <p>To determine whether a user has permission to access a resource or perform an action</p>
  <p>Authorization is the process of determining whether a user or system has the necessary permissions to access a particular resource or perform a particular action. This is distinct from authentication (verifying identity), encryption (protecting data confidentiality), or malware detection (security threat prevention).
