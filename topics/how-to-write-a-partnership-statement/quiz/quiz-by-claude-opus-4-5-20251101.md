# How to write a partnership statement

Question: When writing a partnership statement, what framework should be used to ensure goals are well-defined and actionable?

- [ ] SWOT (Strengths, Weaknesses, Opportunities, Threats)
- [ ] RACI (Responsible, Accountable, Consulted, Informed)
- [ ] SMART (Specific, Measurable, Achievable, Relevant, Timely)
- [ ] MoSCoW (Must have, Should have, Could have, Won't have)

<details>
  <summary>Answer</summary>
  <p>SMART (Specific, Measurable, Achievable, Relevant, Timely)</p>
  <p>Partnership statements should articulate shared goals that are SMART: Specific, Measurable, Achievable, Relevant, and Timely. This framework ensures that partnership objectives are clear, trackable, realistic, aligned with the partnership's purpose, and have defined timeframes for completion.</p>
</details>
