# Progress indicator

Question: What is the primary purpose of a progress indicator in user interface design?

- [ ] To decorate the interface with animated visual elements
- [ ] To provide feedback that communicates the status or completion of a process to the user
- [ ] To prevent users from interacting with the application during loading
- [ ] To display advertising content while users wait for tasks to complete

<details>
  <summary>Answer</summary>
  <p>To provide feedback that communicates the status or completion of a process to the user</p>
  <p>Progress indicators are visual elements that communicate progress or status to users, providing feedback that something is happening. They help users understand expected completion time, enhance user experience by providing a sense of control, and improve usability by helping users gauge time and effort required to complete a task.</p>
</details>
