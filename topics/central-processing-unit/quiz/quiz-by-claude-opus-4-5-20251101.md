# Central Processing Unit (CPU)

Question: What is the primary function of the CPU cache in a computer system?

- [ ] To permanently store the operating system files
- [ ] To store frequently accessed data and instructions to reduce memory access latency
- [ ] To increase the number of cores available for processing
- [ ] To regulate the clock speed of the processor

<details>
  <summary>Answer</summary>
  <p>To store frequently accessed data and instructions to reduce memory access latency</p>
  <p>CPU cache is a small but fast memory unit that stores frequently accessed data and instructions. This helps reduce the time the CPU spends waiting for data from slower main memory (RAM), significantly improving overall system performance.</p>
</details>
