# PARIS matrix

Question: What does PARIS stand for in the PARIS matrix, a project management tool used to clarify roles and responsibilities?

- [ ] Plan, Assign, Review, Implement, Support
- [ ] Participate, Approve, Responsible, Input, Sign-off
- [ ] Primary, Accountable, Resource, Inform, Stakeholder
- [ ] Project, Authority, Resource, Information, Schedule

<details>
  <summary>Answer</summary>
  <p>Participate, Approve, Responsible, Input, Sign-off</p>
  <p>The PARIS matrix is a variation of a Responsibility Assignment Matrix (RAM) where each letter represents a specific level of responsibility: Participate (contributes to completion), Approve (authority to approve/reject work), Responsible (accountable for completing the task), Input (provides feedback and guidance), and Sign-off (authority to confirm completion meets quality standards).</p>
</details>
