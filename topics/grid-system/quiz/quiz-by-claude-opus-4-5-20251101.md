# Grid system

Question: What is the primary purpose of a grid system in design?

- [ ] To add decorative borders around content elements
- [ ] To automatically generate color schemes for layouts
- [ ] To organize and align elements within a layout for balance, hierarchy, and consistency
- [ ] To compress image files for faster loading

<details>
  <summary>Answer</summary>
  <p>To organize and align elements within a layout for balance, hierarchy, and consistency</p>
  <p>A grid system is a structural framework of horizontal and vertical lines that helps designers organize content, create visual alignment, and maintain consistency across a layout. It provides designated spaces for placing elements like images and text, facilitates responsive design adaptation, and streamlines the design process while improving readability and usability.</p>
</details>
