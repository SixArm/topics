# Database Administrator

Question: What is the primary responsibility of a Database Administrator (DBA) regarding data protection and business continuity?

- [ ] Designing user interfaces for database applications
- [ ] Writing application code that interacts with the database
- [ ] Developing and implementing backup and recovery strategies to safeguard data against failures
- [ ] Creating marketing reports from database analytics

<details>
  <summary>Answer</summary>
  <p>Developing and implementing backup and recovery strategies to safeguard data against failures</p>
  <p>A core DBA responsibility is ensuring data can be recovered in case of system failures, errors, or disasters. DBAs regularly perform backups, test restore processes, and maintain disaster recovery plans to ensure business continuity and protect organizational data assets.</p>
</details>
