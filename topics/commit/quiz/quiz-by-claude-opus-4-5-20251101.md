# Commit

Question: What is the primary purpose of a commit in Git?

- [ ] To delete files from the repository permanently
- [ ] To merge two branches into one
- [ ] To create a snapshot of changes and record them with a descriptive message
- [ ] To push changes to a remote server

<details>
  <summary>Answer</summary>
  <p>To create a snapshot of changes and record them with a descriptive message</p>
  <p>A commit in Git is a snapshot of changes to a project that has been saved to the repository. Each commit includes a unique identifier hash and a message describing the changes, enabling developers to track the evolution of the codebase, understand when and why changes were made, and revert to previous versions if necessary.</p>
</details>
