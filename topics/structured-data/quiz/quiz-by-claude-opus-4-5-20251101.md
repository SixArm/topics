# Structured data

Question: What is the defining characteristic of structured data?

- [ ] It is stored exclusively in cloud-based systems
- [ ] It is organized in a predefined format such as rows and columns
- [ ] It requires machine learning algorithms to process
- [ ] It cannot be queried using traditional database tools

<details>
  <summary>Answer</summary>
  <p>It is organized in a predefined format such as rows and columns</p>
  <p>Structured data is formatted data that follows a predefined schema, typically organized into rows and columns. This tidy formatting makes it more readily analyzed than unstructured data because it can be easily searched, filtered, and processed using standard database query languages and analytical tools.</p>
</details>
