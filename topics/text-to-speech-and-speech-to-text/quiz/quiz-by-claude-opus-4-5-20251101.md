# Text-To-Speech (TTS) and Speech-To-Text (STT)

Question: What is another common name for Speech-To-Text (STT) technology?

- [ ] Voice synthesis technology
- [ ] Natural language processing (NLP)
- [ ] Automatic speech recognition (ASR)
- [ ] Audio encoding system (AES)

<details>
  <summary>Answer</summary>
  <p>Automatic speech recognition (ASR)</p>
  <p>Speech-To-Text (STT) is also known as automatic speech recognition (ASR). STT technology converts spoken words into written text by analyzing audio input and using speech recognition algorithms, acoustic modeling, language modeling, and machine learning algorithms to transcribe speech accurately.</p>
</details>
