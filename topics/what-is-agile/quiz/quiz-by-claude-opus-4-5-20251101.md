# What is agile?

Question: What does agile methodology prioritize over following a fixed plan?

- [ ] Comprehensive documentation
- [ ] Contract negotiation
- [ ] Responding to change
- [ ] Processes and tools

<details>
  <summary>Answer</summary>
  <p>Responding to change</p>
  <p>Agile methodology emphasizes responding to change over following a fixed plan. This is one of the four core values from the Agile Manifesto, which enables teams to adapt quickly to changing requirements and deliver value more effectively by course-correcting early rather than rigidly adhering to an initial plan.</p>
</details>
