# Relational database

Question: What mechanism does a relational database use to establish relationships between tables?

- [ ] Indexes that optimize query performance
- [ ] Constraints that limit field values
- [ ] Foreign keys that reference primary keys in other tables
- [ ] Primary keys that uniquely identify records

<details>
  <summary>Answer</summary>
  <p>Foreign keys that reference primary keys in other tables</p>
  <p>In a relational database, relationships between tables are established using foreign keys, which are fields in one table that refer to the primary key in another table. This mechanism enables the database to link related data across multiple tables while maintaining referential integrity.</p>
</details>
