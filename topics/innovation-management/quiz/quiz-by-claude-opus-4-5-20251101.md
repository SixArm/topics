# Innovation management

Question: What is the correct sequence of key phases in the innovation management lifecycle?

- [ ] Evaluation → Resource Allocation → Idea Generation → Implementation
- [ ] Idea Generation → Idea Selection → Resource Allocation → Development → Implementation → Evaluation
- [ ] Resource Allocation → Development → Idea Selection → Evaluation
- [ ] Implementation → Idea Generation → Development → Idea Selection

<details>
  <summary>Answer</summary>
  <p>Idea Generation → Idea Selection → Resource Allocation → Development → Implementation → Evaluation</p>
  <p>Innovation management follows a logical progression: first generating ideas, then selecting the most promising ones based on criteria like feasibility and market potential, allocating necessary resources, developing the innovation through prototyping and testing, implementing it into operations, and finally evaluating its impact and identifying improvements. This structured lifecycle ensures innovations are systematically nurtured from concept to commercialization.</p>
</details>
