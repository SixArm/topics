# Network protocols

Question: Which protocol suite forms the foundational communication framework of the internet and includes TCP, IP, and UDP?

- [ ] SSL/TLS
- [ ] TCP/IP
- [ ] HTTP
- [ ] SMTP

<details>
  <summary>Answer</summary>
  <p>TCP/IP</p>
  <p>TCP/IP (Transmission Control Protocol/Internet Protocol) is the most widely used protocol suite for networking. It includes TCP for reliable data transmission, IP for addressing and routing, and UDP for connectionless communication. This suite forms the fundamental framework that connects devices over the internet.</p>
</details>
