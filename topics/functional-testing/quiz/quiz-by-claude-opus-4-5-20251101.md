# Functional testing

Question: What is the primary purpose of functional testing?

- [ ] To measure the speed and performance of the application under load
- [ ] To examine the internal code structure and logic paths
- [ ] To verify that applications perform their intended functions correctly against specified requirements
- [ ] To identify security vulnerabilities and potential exploits

<details>
  <summary>Answer</summary>
  <p>To verify that applications perform their intended functions correctly against specified requirements</p>
  <p>Functional testing focuses on verifying that software behavior meets specified requirements, ensuring features work as expected from an end-user perspective. It examines what the system does rather than how it does it internally, validating that user-facing functionality operates correctly.</p>
</details>
