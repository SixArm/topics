# Agile Manifesto 2: Working Software

Question: What does the Agile principle "Working software over comprehensive documentation" primarily emphasize?

- [ ] All documentation should be eliminated to maximize development speed
- [ ] Functional, tested software should take precedence over exhaustive written specifications
- [ ] Documentation must be completed before any coding begins
- [ ] Comprehensive technical manuals are required for every software release

  <summary>Answer</summary>
  <p>Functional, tested software should take precedence over exhaustive written specifications</p>
