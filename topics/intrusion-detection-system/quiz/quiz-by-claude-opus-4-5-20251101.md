# Intrusion Detection System (IDS)

Question: What are the two main classification categories of Intrusion Detection Systems?

- [ ] Signature-based and behavior-based
- [ ] Active and passive
- [ ] Network-based and host-based
- [ ] Real-time and batch processing

<details>
  <summary>Answer</summary>
  <p>Network-based and host-based</p>
  <p>IDS systems are classified into two main categories: network-based IDS, which monitors network traffic at the network layer for suspicious activity like scanning and reconnaissance, and host-based IDS, which monitors system and application logs on individual hosts for signs of unauthorized access or unusual activity.</p>
</details>
