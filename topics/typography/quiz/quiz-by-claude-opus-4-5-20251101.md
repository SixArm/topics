# Typography

Question: What is the primary purpose of establishing a visual hierarchy in typography?

- [ ] To ensure all text elements appear identical in size and weight
- [ ] To maximize the number of different fonts used in a design
- [ ] To guide the reader's attention among headings, subheadings, body text, and other elements
- [ ] To eliminate the need for line spacing and letter spacing

<details>
  <summary>Answer</summary>
  <p>To guide the reader's attention among headings, subheadings, body text, and other elements</p>
  <p>Typography helps establish a visual hierarchy by using different font sizes, weights, and styles to direct the reader's focus and attention through the content in a logical order. This makes information easier to scan, understand, and navigate, which is essential for effective communication in user interfaces, documentation, and digital products.</p>
</details>
