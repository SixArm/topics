# Screen reader testing

Question: What is the primary purpose of screen reader testing in digital accessibility?

- [ ] To verify that all images have high-resolution displays across different screen sizes
- [ ] To ensure that content can be perceived, navigated, and interacted with by users who rely on screen readers
- [ ] To optimize website loading speed for mobile devices
- [ ] To check that color contrast meets branding guidelines

<details>
  <summary>Answer</summary>
  <p>To ensure that content can be perceived, navigated, and interacted with by users who rely on screen readers</p>
  <p>Screen reader testing evaluates whether a digital interface properly announces elements like headings, links, buttons, form controls, and landmarks, and verifies that users can navigate and interact with the interface using keyboard or voice commands. This ensures accessibility for individuals who depend on screen readers to access content.</p>
</details>
