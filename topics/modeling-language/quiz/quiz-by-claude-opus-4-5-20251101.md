# Modeling language

Question: What is the primary purpose of a modeling language?

- [ ] To compile source code into executable programs
- [ ] To provide a structured and standardized way to describe and represent aspects of a system or domain
- [ ] To encrypt data for secure transmission over networks
- [ ] To automate deployment of applications to cloud infrastructure

<details>
  <summary>Answer</summary>
  <p>To provide a structured and standardized way to describe and represent aspects of a system or domain</p>
  <p>A modeling language is a formal language that provides a structured and standardized way to express complex concepts, relationships, and behaviors. This enables technology professionals to visualize, document, and communicate system designs across software engineering, system design, and process planning.</p>
</details>
