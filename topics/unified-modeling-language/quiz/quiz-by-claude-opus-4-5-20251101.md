# Unified Modeling Language (UML)

Question: What is the primary purpose of Unified Modeling Language (UML)?

- [ ] To compile and execute software code automatically
- [ ] To replace written documentation in software projects
- [ ] To visualize and communicate the structure, behavior, and relationships of software system components
- [ ] To generate database schemas from natural language descriptions

<details>
  <summary>Answer</summary>
  <p>To visualize and communicate the structure, behavior, and relationships of software system components</p>
  <p>UML is a standardized visual notation that helps developers, architects, and other stakeholders communicate and visualize the structure, behavior, and relationships of different components in a software system. It provides a common language for modeling software systems through various diagram types including class diagrams, sequence diagrams, use case diagrams, and others.</p>
</details>
