# Regression testing

Question: What is the primary purpose of regression testing?

- [ ] To test new features before they are added to the codebase
- [ ] To measure the performance and speed of an application under load
- [ ] To ensure that changes or updates do not introduce new bugs in previously working functionality
- [ ] To verify that software meets user interface design specifications

<details>
  <summary>Answer</summary>
  <p>To ensure that changes or updates do not introduce new bugs in previously working functionality</p>
  <p>Regression testing specifically aims to verify that existing functionalities continue working as intended after code changes, updates, or bug fixes have been made. It involves retesting the system to catch any unintended side effects that new changes may have introduced to previously working features.</p>
</details>
