# Agile Principles

Question: According to Agile principles, what is the primary measure of progress in software development?

- [ ] Comprehensive documentation
- [ ] Working software
- [ ] Number of completed tasks
- [ ] Lines of code written

<details>
  <summary>Answer</summary>
  <p>Working software</p>
  <p>Agile Principle #2 states that "Working software is the primary measure of progress." This reflects the Agile philosophy of valuing tangible, functional results over traditional metrics like documentation volume, task counts, or code quantity. Working software demonstrates real value delivered to customers and stakeholders.</p>
</details>
