# User flow

Question: What is the primary purpose of mapping user flows in UX design?

- [ ] To create visually appealing documentation for stakeholders
- [ ] To identify pain points, bottlenecks, and areas where users might get confused or frustrated
- [ ] To determine the color scheme and typography of the interface
- [ ] To calculate the technical performance metrics of the application

<details>
  <summary>Answer</summary>
  <p>To identify pain points, bottlenecks, and areas where users might get confused or frustrated</p>
  <p>User flows help designers understand and optimize the user experience by revealing where users encounter difficulties. By mapping out the sequence of screens, actions, and decision points, designers can spot usability issues and opportunities for improvement, ultimately reducing friction and guiding users toward their goals more effectively.</p>
</details>
