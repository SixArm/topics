# Iconography

Question: What is a primary benefit of following established conventions and standards in iconography for user interface design?

- [ ] It allows designers to use more complex and detailed imagery
- [ ] It increases the file size of icon assets for better resolution
- [ ] It ensures consistency and familiarity, making icons widely recognized and understood
- [ ] It requires users to learn new visual languages for each application

<details>
  <summary>Answer</summary>
  <p>It ensures consistency and familiarity, making icons widely recognized and understood</p>
  <p>Iconography follows established conventions and standards specifically to ensure consistency and familiarity across applications. Commonly used icons like the hamburger menu for navigation or social media logos have become widely recognized because they adhere to these standards. This consistency reduces the learning curve for users and enables quick, efficient communication regardless of the specific application or platform.</p>
</details>
