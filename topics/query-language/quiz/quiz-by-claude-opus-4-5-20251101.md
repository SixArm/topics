# Query language

Question: What is the primary purpose of a query language?

- [ ] To design the visual interface of database management systems
- [ ] To compile high-level programming code into machine language
- [ ] To interact with databases and retrieve specific information without needing to understand underlying implementation details
- [ ] To encrypt and secure data transmissions between servers

<details>
  <summary>Answer</summary>
  <p>To interact with databases and retrieve specific information without needing to understand underlying implementation details</p>
  <p>A query language is a specialized programming language that allows users to query, manipulate, and manage data stored in a database. It abstracts away the complexity of the underlying database structure, enabling users to focus on what data they need rather than how it is stored or organized.</p>
</details>
