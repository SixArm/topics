# Test data

Question: What is the primary purpose of using test data (test set) in machine learning?

- [ ] To train the model on as much data as possible
- [ ] To provide an unbiased estimate of model performance on unseen data
- [ ] To store the model's hyperparameters for future use
- [ ] To increase the speed of the training process

  <p>To provide an unbiased estimate of model performance on unseen data</p>
