# Quantitative data

Quantitative data is numerical information that can be measured, counted, and analyzed statistically to identify trends, patterns, and averages. It answers "how many," "how much," or "how often," and is typically collected through structured methods like surveys, polls, and experiments. Key examples include height, weight, revenue, and temperature. 

Key Aspects of Quantitative Data

- Definition: Objective, concrete facts expressed in numbers.

- Discrete Data: Whole numbers that are counted (e.g., number of students, items sold).

- Continuous Data: Measurements that can take any value within a range (e.g., length, time, temperature).

- Collection Methods: Close-ended surveys, experiments, sensor readings, and database queries.

- Analysis: Utilizes statistical methods like mean, median, mode, and standard deviation.

- Visualization: Often represented in graphs, charts, and tables. 

Quantitative vs. Qualitative Data

- While quantitative data provides numerical, objective facts, qualitative data provides descriptive, non-numerical insights (words, meanings)
