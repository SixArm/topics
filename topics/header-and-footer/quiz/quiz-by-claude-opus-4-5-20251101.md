# Header and footer

Question: What is the primary purpose of maintaining consistent headers and footers across all pages of a website?

- [ ] To increase the file size of each page for better server performance
- [ ] To display different branding on each page
- [ ] To help users navigate easily and find information regardless of their location within the website
- [ ] To reduce the need for search engine optimization

<details>
  <summary>Answer</summary>
  <p>To help users navigate easily and find information regardless of their location within the website</p>
  <p>Consistency is a key aspect of headers and footers. By appearing uniformly across all pages, they provide users with familiar navigation elements and visual anchors, making it easier to move through the site and locate information without confusion or disorientation.</p>
</details>
