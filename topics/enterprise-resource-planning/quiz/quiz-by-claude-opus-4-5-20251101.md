# Enterprise Resource Planning (ERP)

Question: What is a key advantage of implementing an ERP system in an organization?

- [ ] It requires minimal training and can be deployed instantly
- [ ] It enables organizations to break down functional silos and improve cross-functional collaboration
- [ ] It eliminates the need for a centralized database
- [ ] It restricts data access to individual departments for better security

<details>
  <summary>Answer</summary>
  <p>It enables organizations to break down functional silos and improve cross-functional collaboration</p>
  <p>ERP systems provide a centralized platform where all business units and departments have access to the same data and can work together. This breaks down traditional functional silos, enabling better cross-functional collaboration, improved efficiency, reduced costs, and better-informed decision-making through timely and accurate information sharing across the organization.</p>
</details>
