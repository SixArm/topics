- [ ] A method for optimizing application performance through caching
- [ ] The perceived qualities of an object that suggest how it can be interacted with
- [ ] A testing framework for validating user interface components
- [ ] The process of converting user requirements into technical specifications

  <summary>Answer</summary>
  <p>The perceived qualities of an object that suggest how it can be interacted with</p>
  <p>Affordance refers to the relationship between an object and its potential actions or uses.
