# Communication styles

Question: Which communication style dimension describes the difference between relying on contextual cues, nonverbal signals, and shared cultural understanding versus using explicit verbal statements to convey meaning?

- [ ] Individualistic versus Collectivist Communication
- [ ] High-Context versus Low-Context Communication
- [ ] Formal versus Informal Communication
- [ ] Direct versus Indirect Communication
