# Markup language

Question: What is the primary purpose of a markup language?

- [ ] To compile source code into executable programs
- [ ] To encrypt data for secure transmission
- [ ] To annotate text to define its structure, presentation, or semantics
- [ ] To manage database transactions and queries

<details>
  <summary>Answer</summary>
  <p>To annotate text to define its structure, presentation, or semantics</p>
  <p>A markup language uses special tags or codes embedded within text to indicate how elements should be displayed or processed. These tags provide instructions to software applications or rendering engines on how to handle the content, enabling consistent formatting and organization across different systems and platforms.</p>
</details>
