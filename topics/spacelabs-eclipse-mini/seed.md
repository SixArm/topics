# Spacelabs Eclipse Mini

The Spacelabs Eclipse Mini is a wireless, patch-based Holter recorder designed for continuous cardiac monitoring for up to 30 days. It is primarily used to detect infrequent arrhythmias, such as atrial fibrillation, and is intended for both paediatric and adult patients in clinical or home settings. 

Key Features:
- Extended Recording: Supports high-resolution, 3-channel ECG recording for up to 30 days, providing full disclosure for detailed analysis.
- Integrated Design: The battery is built directly into the adhesive electrode patch, eliminating the need for manual battery insertion or external lead wires.
- Patient Comfort: It is ultra-lightweight (under 15g) and waterproof (IP67), allowing patients to shower and maintain a normal lifestyle during the monitoring period.
- Symptom Tracking: Patients can document symptoms via an event button on the device or through a linked smartphone app.
- Rapid Analysis: Data is downloaded via a standard USB-C cable and can be quickly analyzed using the LifeScreen Pro event screening system or the Pathfinder SL Holter Analysis System.
