# Empathy map - benefits

Question: What is a primary benefit of using an empathy map in design processes?

- [ ] It eliminates the need for user testing by predicting all user behaviors accurately
- [ ] It shifts focus from assumptions and biases to understanding users' actual experiences, promoting user-centered design
- [ ] It automates the design workflow by generating UI components based on user data
- [ ] It reduces project timelines by replacing stakeholder meetings with automated reports

<details>
  <summary>Answer</summary>
  <p>It shifts focus from assumptions and biases to understanding users' actual experiences, promoting user-centered design</p>
  <p>An empathy map promotes user-centered design by helping teams move beyond their own assumptions and biases. By organizing information about what users think, feel, say, and do, designers gain insights based on real user needs rather than speculation, leading to more effective and meaningful design decisions.</p>
</details>
