# Card sorting

Question: What is the key difference between open card sorting and closed card sorting in user research?

- [ ] Open card sorting uses physical cards while closed card sorting uses digital cards
- [ ] In open card sorting, participants create their own categories; in closed card sorting, participants sort cards into predefined categories
- [ ] Open card sorting is done remotely while closed card sorting requires in-person sessions
- [ ] Closed card sorting collects more data than open card sorting

<details>
  <summary>Answer</summary>
  <p>In open card sorting, participants create their own categories; in closed card sorting, participants sort cards into predefined categories</p>
  <p>Open card sorting allows participants to group cards based on their own mental models and create their own category labels, which helps discover how users naturally organize information. Closed card sorting provides predefined categories, making it useful for testing an existing information architecture or evaluating whether predetermined categories match user expectations.</p>
</details>
