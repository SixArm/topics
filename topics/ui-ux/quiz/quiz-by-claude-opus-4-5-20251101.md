# UI/UX

Question: What is the primary distinction between User Interface (UI) and User Experience (UX) design?

- [ ] UI focuses on backend functionality while UX focuses on frontend code
- [ ] UI focuses on visual elements and interactive components while UX focuses on the overall experience and user satisfaction
- [ ] UI and UX are identical terms used interchangeably in the industry
- [ ] UI focuses on user research while UX focuses on typography and icons

<details>
  <summary>Answer</summary>
  <p>UI focuses on visual elements and interactive components while UX focuses on the overall experience and user satisfaction</p>
  <p>UI (User Interface) refers to the visual elements, layout, and interactive components such as buttons, menus, forms, icons, and typography. UX (User Experience) encompasses the overall experience and satisfaction users have while interacting with a product, involving user research, information architecture, interaction design, and usability testing. While distinct, they work together to create an optimal user experience.</p>
</details>
