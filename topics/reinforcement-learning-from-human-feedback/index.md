# Reinforcement Learning from Human Feedback (RLHF)

Reinforcement Learning from Human Feedback (RLHF) is a machine learning technique that aligns large language models with human preferences, making them more helpful, honest, and harmless. While base models are trained on massive datasets to predict the next word, RLHF teaches them to be conversational and follow specific human instructions. The core idea is straightforward: human evaluators rank model outputs, and the model learns to favor responses that people prefer.

The RLHF pipeline operates in three phases. First, supervised fine-tuning trains the base model on high-quality human-written prompt-response pairs so it learns the format of a helpful assistant. Second, reward modeling generates several candidate responses for a single prompt and has human annotators rank them from best to worst; this ranking data trains a separate reward model that can mathematically predict which response a human would prefer. Third, reinforcement learning has the fine-tuned model generate new answers that the reward model scores, and algorithms like Proximal Policy Optimization update the model's parameters to maximize those scores.

RLHF captures subjective qualities like politeness and conciseness that are difficult to encode as hard rules, and it significantly reduces toxic outputs and hallucinations compared to raw base models. However, the full training loop is resource-intensive, requiring multiple models to run simultaneously. A known risk called reward hacking can also emerge, where the model finds shortcuts to score well without genuinely improving, such as producing overly verbose or sycophantic responses.

Newer methods like Direct Preference Optimization (DPO) skip the separate reward model and reinforcement learning loop entirely, optimizing directly on preference pairs. DPO is more stable and less computationally expensive, making it an increasingly popular alternative for many practitioners.
