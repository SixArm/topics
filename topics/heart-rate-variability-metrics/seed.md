# Heart rate variability metrics

Heart rate variability metrics from ECG and PPG data. It offers comprehensive time-domain, frequency-domain, and non-linear analysis, with features for preprocessing data, window-based analysis, and live HRV calculations for embedded systems. 

Key Features of cardio-rs:
- Time-Domain Metrics: Calculates metrics such as SDNN, RMSSD, PNN50, SDSD, and mean heart rate.
- Frequency-Domain Metrics: Computes LF, HF, and VLF using Welch's method.
- Geometric and Non-linear Analysis: Includes Triangular Index, TINN, SampEn, DFA, and LZC.
- Preprocessing: Includes tools for filtering and denoising, as well as peak detection for ECG/PPG data.
- Real-time Analysis: Supports live analysis, making it suitable for wearable devices.
