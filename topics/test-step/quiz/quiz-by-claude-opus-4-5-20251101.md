# Test step

Question: What is a test step in an automated software testing framework?

- [ ] A document that describes the overall testing strategy for a project
- [ ] A complete end-to-end test scenario that validates an entire user workflow
- [ ] The smallest executable unit that encapsulates a specific action or verification
- [ ] A test environment configuration file used to set up automation tools

<details>
  <summary>Answer</summary>
  <p>The smallest executable unit that encapsulates a specific action or verification</p>
  <p>A test step represents the fundamental building block in test automation, performing a single atomic action such as clicking a button, entering text, or validating expected content. This granular approach enables reusability, easier maintenance, better debugging, and supports data-driven testing scenarios.</p>
</details>
