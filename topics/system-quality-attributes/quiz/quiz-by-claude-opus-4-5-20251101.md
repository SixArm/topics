# System quality attributes

Question: Which system quality attribute primarily addresses the system's ability to handle increased user load, transaction volume, or data growth without performance degradation?

- [ ] Maintainability
- [ ] Scalability
- [ ] Compatibility
- [ ] Usability

<details>
  <summary>Answer</summary>
  <p>Scalability</p>
  <p>Scalability refers to the system's ability to handle growth in the number of users, transactions, or data volumes. A scalable system can adapt to changes in demand without experiencing a decline in performance, making it essential for systems that anticipate growth or variable workloads.</p>
</details>
