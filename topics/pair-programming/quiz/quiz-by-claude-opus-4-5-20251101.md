# Pair programming

Question: In pair programming, what are the two distinct roles that programmers alternate between?

- [ ] Coder and tester
- [ ] Driver and navigator
- [ ] Writer and reviewer
- [ ] Leader and follower

<details>
  <summary>Answer</summary>
  <p>Driver and navigator</p>
  <p>In pair programming, the driver is responsible for writing the code while the navigator reviews and guides the driver's work. The two programmers switch roles regularly to share responsibilities and maintain engagement. This collaboration model ensures real-time code review and helps catch errors early.</p>
</details>
