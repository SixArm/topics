# Tom's Opinionated Markup Language (TOML)

Question: What is TOML primarily designed to be used for?

- [ ] A general-purpose programming language for web development
- [ ] A configuration file format for storing application settings
- [ ] A database query language for relational databases
- [ ] A markup language for creating web page layouts

<details>
  <summary>Answer</summary>
  <p>A configuration file format for storing application settings</p>
  <p>TOML (Tom's Opinionated Markup Language) was created by Tom Preston-Werner, co-founder of GitHub, specifically as a configuration file format. It is designed to be easy to read and write for both humans and machines, using key-value pairs to store configuration data. TOML supports various data types including strings, integers, floats, booleans, and arrays, making it well-suited for application configuration files.</p>
</details>
