# Spacelabs LifeScreen Pro

The Spacelabs LifeScreen Pro is a clinical software platform developed by Spacelabs Healthcare for rapid triage and analysis of long-term ambulatory ECG data. It targets cardiology departments and monitoring services that need to process extended recordings, supporting up to 30 days of continuous data captured by devices such as the Eclipse Mini and Eclipse Pro.

The system's primary strength is speed. It analyzes millions of heartbeats in minutes rather than hours, enabling clinicians to identify major arrhythmias shortly after a recording device is returned. Automated detection algorithms identify and quantify key cardiac events including atrial fibrillation, ventricular and supraventricular tachycardia, bradycardia, and pauses. This automation reduces the manual burden on cardiac technicians and accelerates time to diagnosis.

LifeScreen Pro also supports symptom correlation. Patients can log symptoms during monitoring through a companion smartphone app or the recording device's event button. The software then aligns these patient-reported events with the corresponding ECG rhythm strips, giving clinicians direct insight into whether symptoms coincide with detectable arrhythmias.

For visual analysis, the platform provides an interactive dashboard with scroll and zoom controls, heart rate range graphs, and 30-day event trend views. These tools allow technicians to verify automated findings before generating reports.

The system is built on the Spacelabs Runway architecture, which defines a structured clinical workflow. Data is first downloaded through the Sentinel Cardiology Information Management System. LifeScreen Pro then generates a high-level summary and single-page triage report. When more detailed morphological analysis is needed, specific ECG segments can be exported directly to the Pathfinder SL Holter Analysis System for comprehensive review. This layered approach separates fast screening from deep analysis, improving throughput without sacrificing diagnostic depth.
