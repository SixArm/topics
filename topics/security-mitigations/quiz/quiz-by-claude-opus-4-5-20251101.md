# Security mitigations

Question: Which security mitigation technique focuses on removing or disabling unnecessary features and components that could be exploited by attackers?

- [ ] Input validation
- [ ] Encryption
- [ ] Reduce attack surface
- [ ] Patching

<details>
  <summary>Answer</summary>
  <p>Reduce attack surface</p>
  <p>Reducing the attack surface involves removing or disabling unnecessary features, functions, or components that could potentially be exploited by attackers. By minimizing the number of entry points and potential vulnerabilities in a system, organizations significantly decrease the opportunities for malicious actors to compromise the software.</p>
</details>
