# Agile hackathons

Question: What is the primary benefit of applying agile methodologies to hackathons?

- [ ] Extending the event duration to allow for comprehensive documentation
- [ ] Introducing iterative development cycles and continuous feedback loops
- [ ] Eliminating the need for stakeholder involvement during the event
- [ ] Focusing exclusively on perfecting a single feature before demonstration

<details>
  <summary>Answer</summary>
  <p>Introducing iterative development cycles and continuous feedback loops</p>
  <p>The agile framework enhances traditional hackathons by introducing iterative development cycles, continuous feedback loops, and adaptive planning. This approach ensures teams can deliver functional increments within compressed timeframes while receiving ongoing guidance, rather than pursuing perfect solutions in isolation.</p>
</details>
