# Gantt chart

Question: What is a Gantt chart primarily used to visualize in project management?

- [ ] The organizational hierarchy of team members
- [ ] The budget allocation across departments
- [ ] The progress of a project over time using horizontal bars
- [ ] The risk probability matrix for deliverables

<details>
  <summary>Answer</summary>
  <p>The progress of a project over time using horizontal bars</p>
  <p>A Gantt chart is a horizontal bar chart that displays a timeline of a project divided into tasks. Each task is represented by a horizontal bar spanning its duration, with the bar's position showing start and end dates. This visualization helps project managers plan, track progress, and identify critical path tasks that must be completed on time to keep the project on schedule.</p>
</details>
