# Books about UI/UX

Question: Which book introduced the concept of human-centered design and explores the psychology behind creating intuitive, functional user experiences?

- [ ] "Information Architecture: For the Web and Beyond" by Louis Rosenfeld and Peter Morville
- [ ] "The Design of Everyday Things" by Don Norman
- [ ] "Designing Interfaces" by Jenifer Tidwell
- [ ] "The Elements of User Experience" by Jesse James Garrett

<details>
  <summary>Answer</summary>
  <p>"The Design of Everyday Things" by Don Norman</p>
  <p>Don Norman's seminal book explores the principles of design and usability by delving into the psychology behind human-centered design. It provides foundational insights into creating intuitive and functional user experiences, and introduced concepts like affordances and signifiers that have become fundamental to the UI/UX field.</p>
</details>
