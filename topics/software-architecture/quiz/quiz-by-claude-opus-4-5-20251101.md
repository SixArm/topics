# Software architecture

Question: What is the primary goal of software architecture?

- [ ] To write detailed source code for all system components
- [ ] To create a blueprint that guides development and ensures the system meets business, user, and stakeholder needs
- [ ] To eliminate the need for project managers and business owners
- [ ] To select the programming language for the project

<details>
  <summary>Answer</summary>
  <p>To create a blueprint that guides development and ensures the system meets business, user, and stakeholder needs</p>
  <p>Software architecture provides a high-level design that serves as a blueprint for the development team. It establishes a common understanding among all stakeholders—including developers, project managers, and business owners—about how the system's components interact and how functional and non-functional requirements will be met.</p>
</details>
