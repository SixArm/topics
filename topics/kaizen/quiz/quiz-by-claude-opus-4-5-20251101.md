# Kaizen (continuous improvement)

Question: What is the foundational cyclical process that Kaizen methodology is based on?

- [ ] Define-Measure-Analyze-Improve-Control (DMAIC)
- [ ] Build-Measure-Learn (BML)
- [ ] Plan-Do-Check-Act (PDCA)
- [ ] Observe-Orient-Decide-Act (OODA)

<details>
  <summary>Answer</summary>
  <p>Plan-Do-Check-Act (PDCA)</p>
  <p>Kaizen is based on the PDCA cycle, which provides a systematic framework for continuous improvement: Plan (identify opportunities and develop a change plan), Do (implement the changes), Check (measure results to determine effectiveness), and Act (standardize effective changes or iterate on ineffective ones).</p>
</details>
