# Penetration testing

Question: What is the primary goal of penetration testing?

- [ ] To permanently exploit vulnerabilities in production systems
- [ ] To identify weaknesses in a system's defenses before an attacker can exploit them
- [ ] To install monitoring software on target networks
- [ ] To train employees on proper password management

<details>
  <summary>Answer</summary>
  <p>To identify weaknesses in a system's defenses before an attacker can exploit them</p>
  <p>Penetration testing simulates an attack on computer systems or networks to discover vulnerabilities before malicious actors can exploit them. This proactive approach allows organizations to identify and mitigate security risks, strengthening their overall security posture through remediation of discovered weaknesses.</p>
</details>
