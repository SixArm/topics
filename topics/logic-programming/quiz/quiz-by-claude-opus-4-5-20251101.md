# Logic programming

Question: What are the two primary components of a logic programming language?

- [ ] A compiler and an interpreter
- [ ] A knowledge base and a query processor
- [ ] A parser and a lexer
- [ ] A runtime environment and a debugger

<details>
  <summary>Answer</summary>
  <p>A knowledge base and a query processor</p>
  <p>Logic programming languages have two primary components: a knowledge base, which contains facts and rules that define relationships between objects (predicates), and a query processor, which allows users to ask questions and receive answers by applying logical inference to the knowledge base.</p>
</details>
