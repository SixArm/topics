# Level-Oriented Design (LOD)

Question: What is the primary organizing principle of Level-Oriented Design (LOD) in software architecture?

- [ ] Organizing code around individual programming languages and their specific features
- [ ] Structuring systems around horizontal time-based development phases
- [ ] Arranging architecture hierarchically around levels of abstraction
- [ ] Grouping components by the team members responsible for them

<details>
  <summary>Answer</summary>
  <p>Arranging architecture hierarchically around levels of abstraction</p>
