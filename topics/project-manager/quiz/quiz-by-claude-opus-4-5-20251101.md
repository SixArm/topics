# Project Manager

Question: What is the primary responsibility of a Project Manager when managing scope, schedule, and budget?

- [ ] Delegating all decisions to senior management without personal involvement
- [ ] Ensuring alignment with objectives, managing changes, and negotiating trade-offs with stakeholders
- [ ] Focusing exclusively on budget reduction regardless of project quality
- [ ] Completing tasks independently without consulting the project team

<details>
  <summary>Answer</summary>
  <p>Ensuring alignment with objectives, managing changes, and negotiating trade-offs with stakeholders</p>
  <p>A Project Manager's core responsibility for scope, schedule, and budget management involves keeping these elements aligned with project objectives and resource utilization, managing any changes that arise, taking corrective actions when needed, and negotiating trade-offs with stakeholders to balance competing constraints.</p>
</details>
