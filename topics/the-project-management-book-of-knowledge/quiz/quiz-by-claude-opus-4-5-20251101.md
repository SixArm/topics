# The Project Management Book of Knowledge

Question: What are the five process groups defined in the Project Management Book of Knowledge (PMBOK) for managing projects from start to finish?

- [ ] Discovery, Design, Development, Testing, Deployment
- [ ] Initiating, Planning, Executing, Monitoring and Controlling, Closing
- [ ] Scoping, Budgeting, Scheduling, Implementing, Reviewing
- [ ] Analysis, Specification, Construction, Validation, Maintenance

<details>
  <summary>Answer</summary>
  <p>Initiating, Planning, Executing, Monitoring and Controlling, Closing</p>
  <p>The PMBOK defines these five process groups as the standard framework for managing projects. Each group contains specific processes that guide project management activities: Initiating defines and authorizes the project, Planning establishes scope and objectives, Executing carries out the work, Monitoring and Controlling tracks progress and manages changes, and Closing formally completes the project.</p>
</details>
