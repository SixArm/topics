# Geography Markup Language (GML)

Question: What organization developed Geography Markup Language (GML) as an open standard for geographical data interoperability?

- [ ] World Wide Web Consortium (W3C)
- [ ] Open Geospatial Consortium (OGC)
- [ ] International Organization for Standardization (ISO)
- [ ] Internet Engineering Task Force (IETF)

<details>
  <summary>Answer</summary>
  <p>Open Geospatial Consortium (OGC)</p>
  <p>GML was developed by the Open Geospatial Consortium (OGC) specifically to support interoperability between different software systems handling geographical data. The OGC is an international consortium that develops open standards for geospatial content and services, making this the most foundational fact for technology professionals working with GML.</p>
</details>
